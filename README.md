# 📊 Bitrix24 Deal Import Tool

Простой веб-инструмент для массового обновления сделок в Bitrix24 из Excel файлов.

## 🚀 Демо

**Открыть инструмент:** [https://toronto2026.github.io/bitrix24-import/](https://toronto2026.github.io/bitrix24-import/)

## ✨ Возможности

- 📁 **Drag & Drop** загрузка Excel файлов
- ⚡ **Прямой импорт** через Bitrix24 REST API
- 📊 **Real-time прогресс** обновления
- 📥 **Скачивание отчёта** в CSV
- 💾 **Безопасное хранение** webhook в браузере
- 🎨 **Красивый интерфейс** с анимациями

## 📋 Формат Excel файла

Файл должен содержать следующие колонки:

| ID    | ФБ ЛІНК РОБОТИ | ДАТА ФБ ПОСТА2 |
|-------|----------------|----------------|
| 32044 | https://facebook.com/post/123 | 2026-02-04 11:43:00 |
| 32052 | https://facebook.com/post/456 | 2026-02-04 10:48:00 |

- **ID** - ID сделки в Bitrix24 (обязательно)
- **ФБ ЛІНК РОБОТИ** - ссылка на Facebook пост
- **ДАТА ФБ ПОСТА2** - дата публикации поста

## 📖 Использование

### 1. Получите Bitrix24 Webhook

1. Откройте Bitrix24
2. Настройки → Разработчикам → Вебхуки
3. Создайте **Входящий вебхук**
4. Выберите права: **CRM** → **Сделки**
5. Скопируйте URL (например: `https://toronto.bitrix24.com/rest/123/token/`)

### 2. Откройте инструмент

Перейдите на: **[https://toronto2026.github.io/bitrix24-import/](https://toronto2026.github.io/bitrix24-import/)**

### 3. Введите Webhook

Вставьте скопированный URL в поле "Настройка подключения"

### 4. Загрузите Excel

Перетащите файл или нажмите "Выбрать файл"

### 5. Начните импорт

Нажмите "🚀 Начать импорт" и наблюдайте за прогрессом в реальном времени

### 6. Скачайте отчёт

После завершения можно скачать детальный CSV отчёт с результатами

## 🔗 Добавление в Bitrix24

### Добавить в левое меню Bitrix24:

1. Откройте **Bitrix24**
2. Нажмите **⚙️ Налаштування** (внизу слева)
3. **Загальні налаштування** → **Налаштування меню лівої панелі**
4. Нажмите **Додати елемент**
5. Заполните:
   - **Название:** `📊 Импорт сделок`
   - **URL:** `https://toronto2026.github.io/bitrix24-import/`
6. Нажмите **Зберегти**

Теперь в левом меню Bitrix24 появится ваш инструмент! 🎉

## 🛡️ Безопасность

- ✅ Webhook хранится **только в вашем браузере** (localStorage)
- ✅ Никакие данные **не передаются на сторонние серверы**
- ✅ Весь код выполняется **локально в браузере**

## 📝 Обновляемые поля

Инструмент обновляет два пользовательских поля в Bitrix24:

- `UF_CRM_1771237231252` - **ФБ ЛІНК РОБОТИ** (ссылка на Facebook пост)
- `UF_CRM_1771248712742` - **ДАТА ФБ ПОСТА2** (дата публикации)

## ⚙️ Настройка для других полей

Если хотите обновлять другие поля, отредактируйте файл `index.html`:

1. Найдите строки с кодами полей:
```javascript
fields['UF_CRM_1771237231252'] = deal['ФБ ЛІНК РОБОТИ'];
fields['UF_CRM_1771248712742'] = deal['ДАТА ФБ ПОСТА2'];
```

2. Замените коды на ваши UF коды

3. Узнать коды полей можно здесь:
```
https://ваш-портал.bitrix24.com/rest/123/token/crm.deal.fields.json
```

## 🐛 Решение проблем

### "Ошибка при обновлении сделки"
- Проверьте правильность webhook URL
- Убедитесь что webhook имеет права на обновление сделок

### "Нет данных для импорта"
- Проверьте что файл содержит колонку **ID**
- Убедитесь что ID не пустые

### "Deal not found"
- Проверьте что ID сделок существуют в Bitrix24
- ID должны быть числами
